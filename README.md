## Signing in and navigating the Pangeo Jupyter environment

Let's now all log on to [the Pangeo platform on https://ocean.pangeo.io!](https://ocean.pangeo.io/hub/user-redirect/git-pull?repo=https%3A%2F%2Fgithub.com%2Foceanhackweek%2Fohw19-tutorial-overview&urlpath=lab%2Ftree%2Fohw19-tutorial-overview)

You'll need an ORCID account to sign in. If you don't have one, you can create it (register) easily at https://orcid.org/signin

Spin up a **SMALL** machine!

We will be using this for all tutorials in OHW19.

It is a pretty straghtforward environment to navigate, just be reminded that many of the standard buttons you see on Jupyter notebook that runs locally on your computer are only found on the scroll menu on the left.
